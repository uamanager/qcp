## Installation

```bash
    # with yarn:
    yarn add qcp
    # with npm:
    npm i --save qcp
```

## Usage

### Import

```typescript
    import {QuickCrypticoProtocol} from 'qcp';
```
or
```javascript
    const QuickCrypticoProtocol = require('qcp').QuickCrypticoProtocol;
```
